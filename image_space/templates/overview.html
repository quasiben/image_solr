Image Space

This application is a demonstration of comparing uploaded images with those crawled by a domain search engine. It is intended to show the possible interaction even though the current data set is small.

Usage

This application is intended to show how an investigator can drop some images onto a webpage and compare with images that have been crawled by the MEMEX tools.

Addtionally, all images have images that have been taken with the same serial number, such as one might find on popular sites such as Flickr or Twitter. Ultimately, the app will give investigators the ability to use the images to intiate a search coupling with other applications that focus on other search items.

To test this app do the following:

  • Download test images
  • Navigate to compare tab and drop or upload and image
  • View images that have been downloaded at the Image Table tab

Data

The data presented in this demo was a crawl of escort ads from open web sites. This data was the result of a crawl of 32M+ images. The data is powered by a product called Image Catalog, which combines Apache Solr, Apache Tika, and Apache OODT to create a powerful and searchable EXIF metadata and Image-to-Text (OCR) repository of image information.

The serial numbers of cameras are referenced using http://www.stolencamerafinder.com and http://cameratrace.com.
